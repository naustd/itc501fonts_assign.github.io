Helloo Everyone, Here I am using Finger Paint Font.

Description on Finger Paint Font:

  • Finger Paint Font was designed by Carrois Apostrophe.
  • This font is began with an experiment by using an paint brushes effects and then it became a real typeface.
  • The Finger Paint font in HTML is mostly used for children's websites and educational content.
  • This font is licensed under the Open Font License. So, you can use them in your products & projects - print or digital, commercical etc.

Process that I used Finger Paint Font in my WebPage:

      1. This Finger Paint font that I was acquried from fonts.google.com
      2. I opened the Finger paint font by on clicking on that font.
      3. Once, I opened a font there is one button on top-right corner called Get font.
      4. After clicking that Get font button there is two buttons on right-side called Get embed Code and Download all button.
      5. I clicked on get embed code button once I opened that there was few options like Web,Android there I selected a web as option.
      6. In that web I selected a radio button called link in that there is html code and CSS classes.
      7. I included that html code in my head tag and that CSS classes in my style sheet.
